# MQTT Client

## MQTT Client 

### OSX Installation

Python 2
```bash
pip install paho-mqtt
```

Python 3
```bash
pip3 install paho-mqtt
```

### Raspbian Installation

Python 2
```bash
sudo pip install paho-mqtt
```

Python 3
```bash
sudo pip3 install paho-mqtt
```
